# DOT Studio UI Guide

Use this skill for navigation and product-usage questions.

## Layout
- Left sidebar: workspace explorer
- Center canvas: performers, Acts, editors, terminals
- Right inspector: selected performer or Act configuration

## Toolbar
- Terminal controls
- Tracking
- Publish
- Theme toggle
- Settings
- `Assistant` button on the right

## Key Interaction Paths
- Click a performer or Act to inspect it
- Double-click an Act to enter its choreography view
- Drag a performer onto an Act to attach it as a participant
- Open Settings to configure providers, models, and MCP servers
- Open the Asset Library to browse registry assets

## Registry and Drafts
- Registry assets are browsed through the Asset Library
- Local drafts are workspace-local authoring state
- The assistant does not have live registry search results
- If asked for a specific registry asset, direct the user to search in the Asset Library
